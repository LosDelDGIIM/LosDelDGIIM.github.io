    ¥begin{ejercicio}
        Para cada $r ¥in ¥mathbb{R}¥setminus¥{-1,1¥}$ consideremos la curva
        $$
        ¥alpha_r : ¥mathbb{R}¥longrightarrow ¥mathbb{R}^2,¥qquad
        ¥alpha_r(t) = (rt+¥cos(t),¥sin(t)).
        $$

        ¥begin{enumerate}
            ¥item[a)] Probar que $¥alpha_r$ es regular y determinar la curvatura de $¥alpha_r$. ¥¥
            
            Para ver que $¥alpha_r$ es regular, tenemos que ver, por definiciﾃｳn de curva regular, que 
            $¥alpha_r'(t) ¥neq 0 ¥quad ¥forall t ¥in ¥R$. Es claro que 
            $$¥alpha'_r(t) = (r - ¥sen t, ¥cos t)$$
            Ahora
            $$¥alpha'_r(t) = 0 ¥iff ¥begin{cases}
                r - ¥sen t = 0 ¥¥
                ¥cos t = 0
            ¥end{cases}$$
            Por la segunda ecuaciﾃｳn, necesariamente $t = ¥pi/2 + k ¥pi$, con $k ¥in ¥Z$. La primera queda entonces
            $$r - ¥sen ¥left( ¥dfrac{¥pi}{2} + k ¥pi ¥right) = 0$$

            y dado que $$¥sen ¥left( ¥dfrac{¥pi}{2} + k ¥pi ¥right) ¥in ¥{-1,1¥} ¥quad ¥forall k ¥in ¥Z$$

            tenemos que 
            $$r - ¥sen ¥left( ¥dfrac{¥pi}{2} + k ¥pi ¥right) = 0 ¥iff ¥begin{cases}
                r - 1 = 0 ¥iff r = 1 ¥quad& k ¥hspace{-0.3cm} ¥mod 2 = 0 ¥¥
                r + 1 = 0 ¥iff r = -1 ¥quad& k ¥hspace{-0.3cm} ¥mod 2 = 1
            ¥end{cases}$$

            Es decir, $¥alpha'_r(t) = 0 ¥iff r ¥in ¥{-1,1¥}$. Teniendo esto ﾃｺltimo en cuenta, es inmediato que 
            $$¥boxed{¥alpha_r ¥text{ es regular para cada } r ¥in ¥R ¥setminus¥{-1,1¥}}$$

            Para obtener la curvatura, fijado $r ¥in ¥R ¥setminus ¥{-1,1¥}$, dado que $¥alpha_r$ es plana, puesto
            que el codominio es $¥R^2$, y regular, podemos usar la fﾃｳrmula vista en teorﾃｭa:
            
            $$k_{¥alpha_r}(t) = ¥dfrac{¥det(¥alpha'_r(t), ¥alpha_r''(t))}{|¥alpha_r'(t)|^3}$$

            Calculamos el numerador, usando que $¥alpha_r''(t) = (- ¥cos t, -¥sen t)$
            $$¥det(¥alpha'_r(t), ¥alpha_r''(t)) = ¥begin{vmatrix}
                r - ¥sen t & -¥cos t ¥¥
                ¥cos t & - ¥sen t
            ¥end{vmatrix} = -(r- ¥sen t)¥sen t + ¥cos^2t =$$
            $$- r ¥sen t + ¥sen^2t + ¥cos^2 t = 1 - r ¥sen t$$
            y el denominador (trabajamos con la norma euclﾃｭdea denotada por $| ¥cdot |$)
            $$|¥alpha_r'(t)|^2 = (r - ¥sen t)^2 + ¥cos^2 t = r^2 - 2r¥sen t + ¥sen^2t + ¥cos^2t = 
            r^2 - 2r ¥sen t + 1 ¥Longrightarrow$$
            $$|¥alpha_r'(t)|^3 = (r^2 - 2r ¥sen t + 1)^{3/2}$$
            y juntando ambas cosas
            $$k_{¥alpha_r}(t) = ¥dfrac{1 - r ¥sen t}{(r^2 - 2r ¥sen t + 1)^{3/2}}$$

            ¥item[b)] Deducir que $k_{¥alpha_r}$ no se anula si y sﾃｳlo si $|r|<1$, y que en ese caso
            $$
            k_{¥alpha_r}¥geq ¥frac{1-|r|}{(|r|+1)^3}.
            $$

            Como el denominador nunca se anula por ser la curva regular, tenemos que 
            $$k_{¥alpha_r}(t) = 0 ¥iff 1 - r ¥sen t = 0 ¥iff ¥sen t = ¥dfrac{1}{r}$$
            Como $¥sen(t) ¥in [-1,1]$ para todo $t ¥in ¥R$, debe ser 
            $$¥dfrac{1}{|r|} = ¥left| ¥dfrac{1}{r} ¥right| ¥leq 1 ¥iff |r| ¥geq 1$$
            Pero $r ¥in ¥R ¥setminus ¥{-1,1¥}$, luego $|r| ¥neq 1$ y
            $$k_{¥alpha_r}(t) = 0 ¥iff |r| > 1$$ 
            y entonces estﾃ｡ claro que
            $$k_{¥alpha_r}(t) ¥text{ no se anula} ¥iff |r| < 1$$
            Para ver que se da la desigualdad del enunciado, supongamos que $|r| < 1$. Usando que 
            $r ¥sen t ¥leq r ¥leq |r|$, para todo $t ¥in ¥R$, vemos que 
            $$r ¥sen t ¥stackrel{(*)}{¥leq} |r| ¥iff - r ¥sen t ¥geq -|r| ¥iff 1 - r ¥sen t ¥geq 1 - |r|$$
            Ahora, hay que acotar superiormente el denominador por $(|r| + 1)^3$. Se deduce directamente de
            la expresiﾃｳn de $¥alpha'_r$ y de la desigualdad triangular
            $$¥alpha'_r(t) = (r - ¥sen t, ¥cos t) ¥Longrightarrow |¥alpha'_r(t)| = 
            ¥sqrt{(r - ¥sen t)^2 + ¥cos^2t} ¥stackrel{(*) + |¥cos t| ¥leq 1}{¥leq} ¥sqrt{|r|^2 + 1} = $$
            $$||r|^2+1| ¥leq ||r|^2| + |1| = |r| + 1 ¥Longrightarrow |¥alpha_r'(t)|^3 ¥leq (|r| + 1)^3$$
            de donde
            $$¥boxed{k_{¥alpha_r}(t) = ¥dfrac{1 - r ¥sen t}{|¥alpha_r'(t)|^3} ¥geq ¥dfrac{1 - |r|}{(|r| + 1)^3} ¥quad ¥forall t ¥in ¥R}$$
        ¥end{enumerate}
    ¥end{ejercicio}

            Estudiamos ahora que se cumpla (¥ref{equation:ec1}). Lo probamos por doble contenido.
            ¥begin{proof}‾
                ¥begin{itemize}
                    ¥item[$¥subseteq)$] Sea $(x,y,z) ¥in F(¥Rmas ¥times ¥R)$. Entonces, existen $(r,¥theta) ¥in ¥Rmas ¥times ¥R$ tal que
                    $(x,y,z) = F(r, ¥theta) := (r ¥cos ¥theta, r ¥sen ¥theta, ¥sqrt{3} r)$. Ahora, tenemos que ver que se cumple que

                    $$3(x^2 + y^2) = 3(r^2 ¥cos^2 ¥theta + r^2 ¥sen^2 ¥theta) = 3(r^2 (¥cos^2 ¥theta + ¥sen^2 ¥theta)) = 3r^2$$
                    y luego el miembro derecho

                    del cono, $z^2 = 3(x^2 + y^2) = 0$, y deberﾃｭa ser $z = 0$, lo cual es una contradicciﾃｳn con que $(x,y,z) ¥in C$. Como $(x,y) ¥neq (0,0)$,

                    ¥qedhere

            $$(r ¥cos ¥theta, r ¥sen ¥theta, ¥sqrt{3}r) = X_s(r, ¥theta) = X_s(¥rho, ¥varphi) = (¥rho ¥cos ¥varphi, ¥rho ¥sen ¥varphi, ¥sqrt{3}¥rho)

            ¥Func{X_s^{-1}}{V_s}{U_s}{(x,y,z)}{X_s^{-1}(x,y,z) = (r, ¥theta)}

            $¥theta$, usamos que $(x,y,z)$ es un punto de $V_s = C ¥setminus R_s^{+}$, luego la direcciﾃｳn de $(x,y)$ no es la de $s$. Existe pues un ﾃｺnico
            ﾃ｡ngulo $¥theta ¥in ¥left]s, s + 2 ¥pi ¥right[$, tal que $(x,y) = r(¥cos ¥theta, ¥sen ¥theta)$. Denotamos a dicho ﾃ｡ngulo $¥theta = ¥arg_s(x,y)$.
            Asﾃｭ, la inversa anterior viene dada por $X_s^{-1}(x,y,z) := (¥sqrt{x^2 + y^2}, ¥arg_s(x,y)) = (r, ¥theta)$. Al ser composiciﾃｳn de

            Falta por ver que la diferencial es inyectiva, lo cual sabemos por teorﾃｭa que equivale a ver que las parciales $(X_s)_r$ y $(X_s)_¥theta$
            son linealmente independientes. En este caso, dado que $X_s(r, ¥theta) = (r ¥cos ¥theta, r ¥sen ¥theta, ¥sqrt{3}r)$, calculamos

            $$(X_s)_¥theta = (-r ¥sen ¥theta, r ¥cos ¥theta, 0)$$

            y $r > 0$, se deduce que $(X_s)_r(r, ¥theta) ¥times (X_s)_¥theta(r, ¥theta) ¥neq 0 ¥quad ¥forall (r, ¥theta) ¥in U_s$. ¥¥
            
            En definitiva, hemos demostrado que $X_s : U_s ¥to ¥R^3$ es una parametrizaciﾃｳn local de $C$, donde el entorno parametrizado es 
            $V_s = C ¥setminus R_s^{+}$.  

            ¥item[b)] Demostrar que existe una ﾃｺnica aplicaciﾃｳn de Gauss $N:C¥to ¥mathbb{S}^2$ tal que
            ¥begin{equation}¥label{equation:ec3}
                N¥circ F=¥frac{F_r¥times F_¥theta}{¥|F_r¥times F_¥theta¥|}.
            ¥end{equation}
            Determinar la expresiﾃｳn local en la carta $X_s$ de $C$, $s¥in¥mathbb{R}$, de la primera forma fundamental, 
            la segunda forma fundamental, el endomorfismo de Weingarten, la curvatura de Gauss, la curvatura media y las 
            curvaturas y direcciones principales (en la direcciﾃｳn de $N$). Determinar para cada $p ¥in ¥R^2 ¥setminus ¥{0¥}$
            y $¥omega ¥in ¥R^2$ la ﾃｺnica geodﾃｩsica en $C$ que pasa por $G(p)$ con velocidad $dG(p) ¥in T_{G(p)} C$. ¥¥

            Nuevamente seguimos el orden de apariciﾃｳn de lo que se pide del apartado. Primero, recordamos del apartado anterior que $X_s = F_{|¥Rmas ¥times ]s, s + 2¥pi[}$,
            luego
            $$F(r,¥theta) = (r ¥cos ¥theta, r ¥sen ¥theta, ¥sqrt{3}r)$$
            $$F_r = (¥cos ¥theta, ¥sen ¥theta, ¥sqrt{3})$$
            $$F_¥theta = (-r ¥sen ¥theta, r ¥cos ¥theta, 0)$$
            $$F_r ¥times F_¥theta = (-¥sqrt{3}r ¥cos ¥theta, -¥sqrt{3} r ¥sen ¥theta, r) $$
            Ahora, 
            $$¥norma{F_r ¥times F_¥theta} = ¥sqrt{(-¥sqrt{3}r ¥cos ¥theta)^2 + (-¥sqrt{3} r ¥sen ¥theta)^2 + r^2} = 
            ¥sqrt{3r^2 + r^2} ¥stackrel{r>0}{=} 2r$$
            luego
            $$¥dfrac{F_r ¥times F_¥theta}{¥norma{F_r ¥times F_¥theta}} = ¥left( -¥dfrac{¥sqrt{3}}{2} ¥cos ¥theta, 
            -¥dfrac{¥sqrt{3}}{2} ¥sen ¥theta, ¥dfrac{1}{2} ¥right)$$

            $$N(F(r,¥theta)) = ¥left( -¥dfrac{¥sqrt{3}}{2} ¥cos ¥theta, -¥dfrac{¥sqrt{3}}{2} ¥sen ¥theta, ¥dfrac{1}{2} ¥right)$$
            Vemos que estﾃ｡ bien definida ya que si $(r,¥theta), (¥rho, ¥varphi) ¥in ¥Rmas ¥times ¥R$, con $F(r,¥theta) = F(¥rho, ¥varphi)$ entonces por lo 
            que ya hemos visto en la demostraciﾃｳn de la inyectividad de $X_s$, $r = ¥rho$ y $(¥cos ¥theta, ¥sen ¥theta) = (¥cos ¥varphi, ¥sen ¥varphi)$,
            luego $$¥theta ¥equiv ¥varphi ¥hspace{-0.3cm} ¥mod 2 ¥pi$$

            Obtenemos ahora la expresiﾃｳn explﾃｭcita de $N(x,y,z)$, con $p = (x,y,z) ¥in C$. Entonces por (¥ref{equation:ec1}), $p = F(r,¥theta)$,
            con $r = ¥sqrt{x^2 + y^2} > 0$ y $¥theta ¥in ¥R$ determinado por $$¥cos ¥theta = ¥dfrac{x}{r} ¥quad ¥sen ¥theta = ¥dfrac{y}{r}$$
            Por tanto, $$N(x,y,z) = ¥left( -¥dfrac{¥sqrt{3}}{2} ¥dfrac{x}{¥sqrt{x^2 + y^2}}, -¥dfrac{¥sqrt{3}}{2} ¥dfrac{y}{¥sqrt{x^2 + y^2}}, 
            ¥dfrac{1}{2} ¥right) = ¥left( -¥dfrac{¥sqrt{3}x}{2¥sqrt{x^2 + y^2}}, -¥dfrac{¥sqrt{3}y}{2¥sqrt{x^2 + y^2}}, 
            ¥dfrac{1}{2} ¥right)$$

            Ahora es fﾃ｡cil ver que $N : C ¥to ¥Ss^2$, pues
            $$¥norma{N(x,y,z)}^2 = ¥dfrac{3x^2}{4(x^2 + y^2)} + ¥dfrac{3y^2}{4(x^2 + y^2)} + ¥dfrac{1}{4} = ¥dfrac{3}{4} ¥dfrac{x^2 + y^2}{x^2 + y^2}
            + ¥dfrac{1}{4} = 1$$

            $N$ es diferenciable tipo c), ya que $N = i ¥circ N : C ¥to ¥R^3$ vista como aplicaciﾃｳn $C ¥to ¥R^3$
            lo es (en el sentido a)), porque sus componentes son restricciones a $C$ de funciones diferenciables definidas en el abierto

            Por construcciﾃｳn, $N$ es normal a la superficie, pues $N$ verifica (¥ref{equation:ec3}), y en particular, $N$ es perpendicular
            al plano tangente de $C$ en cada punto $p ¥in C$, dado que $T_p C = ¥langle (X_s)_r, (X_s)_¥theta ¥rangle$, y $N(p)$ es un producto
            vectorial de estos dos vectores. ¥¥

            Podemos afirmar ya que $N$ es una aplicaciﾃｳn de Gauss. Veamos que es ﾃｺnica. Supongamos que existe otra aplicaciﾃｳn de Gauss
            $¥tilde{N} : C ¥to ¥Ss^2$ cumpliendo (¥ref{equation:ec3}). Queremos ver que $¥tilde{N} = N$. Sea $p ¥in C$. Por (¥ref{equation:ec1}),
            existen $(r, ¥theta) ¥in ¥Rmas ¥times ¥R$ tales que $p = F(r,¥theta)$. Entonces

            Sin embargo, $N$ tambiﾃｩn cumple (¥ref{equation:ec3}), luego
            $$¥dfrac{F_r ¥times F_¥theta}{¥norma{F_r ¥times F_¥theta}}(r,¥theta) = (N ¥circ F)(r, ¥theta) = N(F(r,¥theta)) = N(p)$$
            Como $p ¥in C$ era arbitrario, concluimos que $¥tilde{N} = N$, luego la aplicaciﾃｳn de Gauss es ﾃｺnica. ¥¥

            Obtenemos ahora el resto de cosas que pide el apartado. En la carta $X_s = F_{|U_s}$ usamos las coordenadas $(r, ¥theta)$. 
            La base del plano tangente $T_{X_s(r,¥theta)}C$ es $B = ¥{(X_s)_r, (X_s)_¥theta¥}$. Ya tenemos
            $$(X_s)_r = (¥cos ¥theta, ¥sen ¥theta, ¥sqrt{3})$$
            $$(X_s)_¥theta = (-r ¥sen ¥theta, r ¥cos ¥theta, 0)$$

            La primera forma fundamental sabemos que se define como ¥Func{g_p}{T_p C ¥times T_p C}{¥R}{(v,w)}{g_p(v,w) := ¥prodesc{v}{w}}

            En la carta $X_s$ tomamos $p = X_s(r, ¥theta)$. Ahora, obtenemos igual que en los ejemplos vistos en teorﾃｭa los coeficientes de 
            la primera forma fundamental
            $$E = ¥prodesc{(X_s)_r}{(X_s)_r} = |X_r|^2 = ¥cos^2 ¥theta + ¥sen^2 ¥theta + 3 = 4$$
            $$F = ¥prodesc{(X_s)_r}{(X_s)_¥theta} = - r ¥cos ¥theta ¥sen ¥theta + r ¥cos ¥theta ¥sen ¥theta + 0 = 0$$
            $$G = ¥prodesc{(X_s)_¥theta}{(X_s)_¥theta} = |X_¥theta|^2 = r^2 ¥sen^2 ¥theta + r^2 ¥cos^2 ¥theta = r^2$$

            ¥end{pmatrix} = ¥begin{pmatrix}

            ¥end{pmatrix}}$$

            (¥sqrt{3} ¥sen ¥theta, - ¥sqrt{3} ¥cos ¥theta, 0)$$

            $$(N^{X_s})_¥theta = -¥dfrac{¥sqrt{3}}{2r} (X_s)_¥theta$$

            Por lo tanto

            principal asociada a $k_2 = ¥sqrt{3}/(2r)$. ¥¥
